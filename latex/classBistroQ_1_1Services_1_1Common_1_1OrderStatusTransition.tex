\doxysubsection{Detailed Description}
Manages the status transitions for orders in the restaurant POS system. This class enforces valid state transitions and prevents invalid status updates. 

The class maintains a predefined set of valid status transitions. Current valid transitions are\+:
\begin{DoxyItemize}
\item Pending → In\+Progress, Cancelled
\item In\+Progress → Pending, Completed Any other transitions will result in a Conflict\+Exception. 
\end{DoxyItemize}
