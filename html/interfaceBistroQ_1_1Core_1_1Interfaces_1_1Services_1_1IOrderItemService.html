Task<(IEnumerable< DetailOrderItemDto > Items, int Count)> BistroQ.Core.Interfaces.Services.IOrderItemService.GetOrderItemsAsync ( OrderItemCollectionQueryParams queryParams)

Retrieves a paginated list of order items based on the specified query parameters.

Parameters
queryParamsParameters for filtering, sorting, and pagination of order items.
Returns
A tuple containing:
  • Items: Collection of order items matching the query parameters
  • Count: Total number of items matching the query (before pagination)

The returned count represents the total number of items that match the filter criteria, not just the number of items in the current page.

OrderItemStatus status )

Updates the status of multiple order items simultaneously.

Parameters
orderItemIdsCollection of order item IDs to update.
statusThe new status to apply to all specified order items.
Returns
Collection of updated order items with their new status.

This operation:

  • Updates all specified order items atomically
  • Validates status transitions for each item
  • Returns the updated items in their new state
Exceptions
ConflictExceptionThrown when any status transition is invalid.
ResourceNotFoundExceptionThrown when any of the specified order items cannot be found.
